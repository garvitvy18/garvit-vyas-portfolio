INSTACART BASKET ANALYSIS

LINKS

OVERVIEW

Analyzed 3 million + Instacart orders in PostgreSQL using pure SQL to uncover reorder dynamics, cross-sell opportunities, and peak shopping windows. Advanced SQL techniques—window functions, CTEs, pivots, analytical aggregates—converted raw transaction data into business-ready insights for merchandising, marketing, and operations.

KEY ACHIEVEMENTS

  • Reorder Dynamics — flagged “hero” SKUs (> 75 % reorder) and quantified loyalty by user segment.
  • Product Affinity — built co-occurrence matrices, ranking bundles by lift/support for promotions.
  • Peak Windows — mapped hourly / seasonal spikes to guide staffing & inventory planning.
  • Business Impact — recommendations project +15 % repeat revenue and +8 % AOV.
