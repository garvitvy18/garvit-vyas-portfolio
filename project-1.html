Ring NoC on ESP

Overview

Replaced the default mesh interconnect in Embedded Scalable Platform with a 3-port ring router; Hamiltonian task mapping achieved 12 % area savings and 3 % latency reduction on FPGA at 100 MHz.

Key Results

  • Architected parameterised ring router (SystemVerilog).
  • Integrated with ESP and generated bitstream for Xilinx VCU118.
  • Benchmarked 7 PARSEC workloads → 3 % average latency improvement.
  • Open-sourced RTL & scripts; merged upstream.
