OVERVIEW

End‑to‑end GenAI pipeline that turns any image into a creative, context‑aware Instagram caption. Combines BLIP vision‑language embeddings with GPT‑4 via the OpenAI API, served through an async FastAPI backend and deployed on Heroku.

KEY ACHIEVEMENTS

  • Integrated BLIP on HuggingFace and orchestrated GPT‑4 prompts for rich captions.
  • Designed an async FastAPI service with model caching; sub‑2 s end‑to‑end latency.
  • Deployed via Docker/Heroku using an optimized Procfile & .slugignore for fast rollouts.
